# Results Folder

This folder contains the result artifacts, visual outputs, and summary notes for both orientation-estimation methods used in the project.

## Subfolders
- **accelerometer_method/** — result files for the accelerometer-only method
- **sensor_fusion_method/** — result files for the sensor fusion method

## Typical Contents
The files in this folder may include:
- static and dynamic test plots
- roll, pitch, and yaw visualizations
- 3D orientation animation outputs
- comparison figures
- summary notes interpreting the results

## Purpose
The results in this folder support:
- evaluation of orientation estimation accuracy
- comparison between the two methods
- visualization of motion behavior over time
- interpretation of static and dynamic testing performance

## Notes
The accelerometer-only method serves as a baseline for tilt estimation, while the sensor fusion method provides a more complete and robust solution for full 3D orientation tracking.

Refer to the method-specific subfolders for detailed summary notes and result files.
