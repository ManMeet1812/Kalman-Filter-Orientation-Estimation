# Data Folder

This folder contains CSV data files generated and used during orientation estimation experiments for both methods:

- **accelerometer_method/** — data related to accelerometer-only tilt estimation
- **sensor_fusion_method/** — data related to sensor fusion orientation estimation

## Contents
The data in this folder may include:
- raw or recorded output logs
- processed CSV files generated from embedded output
- sample datasets used for plotting and analysis
- method-specific experiment outputs

## Note
Large raw datasets are not included unless necessary. This repository focuses on documenting the project structure, analysis workflow, and representative result data.

## Purpose
The data files in this folder support:
- angle analysis over time
- graph generation
- 3D orientation visualization
- comparison between accelerometer-only and sensor-fusion-based estimation methods
